# Study Time Analyzer

A beginner-friendly web application designed to help students track and analyze the time they spend studying different subjects.

## Features
- Add subject name and study hours
- View total study time
- Identify the most-studied subject
- Clean and simple interface

## Technologies Used
- HTML
- CSS
- JavaScript

## How to Use
1. Open the project in a browser
2. Enter subject name and study hours
3. Click add to see analysis

## Learning Outcome
- JavaScript arrays
- Basic logic building
- DOM manipulation
